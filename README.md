# Scrapy
This project contains examples of spiders that extract data from different websites.

# Requirements

Python 3.8

# Built With

- PyCharm Community Edition 2019.3.3
- Scrapy 1.8.0

# Installation

Install Scrapy with:
```sh
pip install Scrapy
```

Install User-Agents with:
```sh
pip install scrapy-user-agents
```
